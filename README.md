# framework
Framework – уникальный OSINT инструмент для сбора данных мз открытых источников.

Установка :
1. Скачайте Termux
2. Обновите модули командой
pkg update
pkg upgrade
3. Установите Python командой
pkg install python
4. Установите Git командой
pkg install git
5. Скопируйте репозиторий командой
git clone https://github.com/ikeaprogrammist/framework
6. Установите путь к программе командой
cd framework
7. Запустите программу командой
python framework.py
